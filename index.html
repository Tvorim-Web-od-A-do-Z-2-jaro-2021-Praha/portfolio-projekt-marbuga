Czechitas kurzy

Digitální akademie Testování

Digitální akademie je taková tříměsíční škola základů testování, programování a quality assurance. Člověk se naučí používat základní principy a techniky testování v kontextu agilního vývoje softwaru, analyzovat a interpretovat uživatelské požadavky a srozumitelně reportovat defekty. Taky naplánovat a provést manuální testování webové aplikace, dokumentovat a vyhodnocovat testování. Nakoukne pod pokličku i základům objektového programování v Javě a použití Selenia pro automatizované testování aplikace.

Tvořím Web od A do Z 1

Dlouhodobý kurz tvorby webu byl pro mě prvním kurzem u Czechitas, díky kterému jsem nabyla základní znalosti HTML a CSS a techniky pro tvorbu layoutu. Během 12 týdnů jsme se věnovaly i “mobile first” responzivnímu webdesignu a naučily se používat media queries a responzivní obrázky. Součástí kurzu je i představení zcela nového CSS modulu pro tvorbu layoutů - Flexboxu a práce s ním. Kurz vedl skvělý lektor a i díky němu jsem se rozhodla dál se v této oblasti vzdělávat a přihlásila se na navazující jarní kurz HTML a CSS 2.

Základy datové analytiky

Během desetitýdenního kurzu jsme se naučily, co znamená datová analytika, kde se v praxi používá a co vše znamená práce datového analytika. Ukázaly si datové modely a modelování datových skladů. Velkou část kurzu jsme se věnovaly databázím a jazyku SQL – SELECTy už nám tak nejsou vůbec cizí. Další část kurzu byla věnována práci s nástrojem PowerBI a vizualizaci dat.

Tvořím Web od A do Z 2

V návaznosti na první kurz tvorby webu se nyní účastním zatím posledního kurzu u Czechitas zaměřeného na prohloubení znalostí kódování webu. Postupně procházíme další zajímavé oblasti z této problematiky: CSS grid - nejmodernější přístup ke tvorbě layoutů, metodologie pro psaní CSS - BEM, OCSS, Sass - vylepšené CSS, responzivní obrázky, vektorová grafika na webu, animace, optimalizace webu.
